# Verilog-ethernet - RMII

Test of verilog-ethernet alone. It decodes Ethernet/IP/UDP packets, and echoes them. 

It was implemented on the [PYNQ-Z2 board](../../../../documentation/pynqz2_user_manual_v1_0.pdf) with the [LAN8720 Ethernet physical chip](../../../../documentation/LAN8720.pdf)

The file verilog-ethernet_rmii_dep.list is the list of all necessary files to create a vivado project.
