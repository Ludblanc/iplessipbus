# Verilog-ethernet - MII
Test of verilog-ethernet alone. It decodes Ethernet/IP/UDP packets, and echoes them. 

It was implemented on the [VC707 Evaluation Board](../../../../documentation/readme.md) with the [CN-0506 Ethernet physical chip](../../../../documentation/readme.md)

The file verilog-ethernet_mii_dep.list is the list of all necessary files to create a vivado project.
